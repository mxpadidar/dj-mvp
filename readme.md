# DJANGO MVP STARTER

A minimal Django starter template for building server-rendered MVPs.

## Tech Stack

- Python 3.14+
- Django 6.0+
- `uv` for package management

## Getting Started

```bash
git clone https://github.com/mxpadidar/dj-mvp.git
cd dj-mvp

make install
make migrate
make superuser
make run
```

Open `http://127.0.0.1:8000`.
